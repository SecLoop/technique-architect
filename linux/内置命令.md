






* expect  
`expect是一个免费的编程工具，用来实现自动的交互式任务，而无需人为干预。说白了，expect就是一套用来实现自动交互功能的软件。需要安装expect`
```
#!/usr/bin/expect
set file [lindex $argv 0]   # 获取第一个参数
set timeout 3600
spawn scp $file root@10.252.79.31:/root/gitlab_backup
expect  {
"*password:" { send "Grg@)!*\n" }
}
expect eof
exit
```

* 查看版本当前操作系统内核信息

* 查看当前操作系统版本信息
```
cat /proc/version
```

* 查看版本当前操作系统发行版信息
